## Yulin Characteristic Tourism Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/418ce3dbef124e1cb4aa0f916c8eaf24/ghb20250918) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Tourist Attractions
5. Souvenirs
6. Special Food
7. Travel Journal Information
8. System Announcements
9. Backend Management
10. Message Board
11. Shopping Cart
12. Personal Center
13. My Orders
14. My Addresses
15. My Favorites


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Product Category Management
4. User Management
5. Attraction Category Management
6. Tourist Attraction Management
7. Souvenir Management
8. Special Food Management
9. Travel Journal Information Management
10. System Management
11. Order Management
